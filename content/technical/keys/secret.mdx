title: 私钥和地址
---

由私钥生成地址的过程如下:

- 随机生成私钥
- 由私钥生成公钥
- 公钥压缩成为地址

## 私钥

私钥是随机生成的 256 个比特位, 即随机生成 256 个 0 或 1, 一个字节是 8 个比特位, 所以私钥是 32 个字节。

<PrivateKeyGenerator />

## 公钥

公钥由私钥计算而来。私钥乘以椭圆曲线 `secp256k1` 的基点 G 得到公钥。

<PublicKeyGenerator />

## 地址

由于公钥很长, 为了方便使用, 需要对公钥进行精简, 精简后的结果就是地址。

## WIF

随机生成的私钥不能直接导入钱包, 需要将原始私钥转换为 WIF 私钥。

WIF(`Wallet Import Format`)是一种在钱包之间导入导出时使用的私钥格式。

<WIFPrivateKeyGenerator />

WIF 私钥总是以固定字符开头:

- 主网: 未压缩私钥以 `5` 开头, 压缩私钥以 `K` 或 `L` 开头
- 测试网: 未压缩私钥以 `9` 开头, 压缩私钥以 `c` 或 `t` 开头
